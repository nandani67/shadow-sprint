# 🎮 Shadow Sprint

A minimalist endless runner where your own shadow becomes your biggest obstacle.

## Game Idea
- Genre: Endless Runner
- Player: Glowing cube
- Controls: Tap to jump
- Goal: Survive as long as possible
- Collectibles: Blue crystals
- Obstacles: Spikes, walls, and your own shadow
